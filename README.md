# Smart Parking System for Smart Cities

This project is a smart parking management system designed to make parking in smart cities easier and more efficient. It features a web-based interface for users to book and manage parking tickets, alongside an Arduino-based hardware integration for physical parking slot management.

## Features
- **Web Interface:** Includes pages for index, booking, and ticket management (`booking.html`, `ticket.html`, `index.html`).
- **Arduino Integration:** Contains Arduino sketches (`arduino/smart_parking.ino`) to connect with hardware sensors/modules.
- **Firebase Backend:** Uses Firebase for real-time database and data management (`firebase-config.js`).
- **Dynamic Frontend:** Powered by JavaScript (`script.js`) with responsive styling (`style.css`).

## Technologies Used
- HTML / CSS / JavaScript
- Firebase (Realtime Database / Authentication)
- Arduino (C++)
- Node.js (for package management)

## Setup
1. Clone the repository.
2. Ensure you have the required Firebase credentials updated in `firebase-config.js`.
3. Open `index.html` in your browser to view the application.
4. For hardware setup, flash the `arduino/smart_parking.ino` code onto your Arduino board.
